PA e Razão

PA significa progressão arítmétrica, que por sua vez, significa que é uma sequência, que a partir do segundo número/termo, vai ser o numero anterior + a razão.

Mas o que é uma razão?

Simples, razão é como se fosse a "diferença" entre um e outro. Se lembra lá no ensino fundamental onde tinha que calcular uma subtração? o resultado era a diferença entre os números, certo? Então, vamos supor que a sequência seja (1, 2, 3, 4, 5..) e por aí vai. Qual a diferença entre os números? 1!, significa que a razão é 1, por "pula" de 1 em 1.

Calcular razão

Agora vamos dizer que a a sequencia seja (1, 3, 5, 7) como é de costume, nós conseguimos fazer de cabeça, mas vamos pegar essa sequência que é mais simples, a fórmula para calcular é R = An - An-1, onde R é a razão, que é a diferença, An é alguma posição. para calcular devemos pegar a A2, que é 3, e fazer - A2-1, que A2-1 se lê como se fosse "posição 2 menos 1" que volta a ser a posição 1, que é 1, então R = 3-1 = 2, Razão é 2!
